# Warehouse Management System

A console application written in C# for managing warehouse products.

## Features

- Add products
- Show all products
- Show products from group S
- Search products by code
- Edit products
- Delete products
- Save products to file
- Load products from file

## Product Information

Each product contains:

- Code
- Name
- Quantity
- Unit (kg/l)
- Shelf Life
- Date
- Position
- Group (S/N)
- Temperature (for group S)

## Technologies

- C#
- .NET
- File Handling
- Lists
- Methods
- Classes
